Minimum d'une liste

Description

Etant donnee une liste, retourner la valeur minimal parmis toutes les valeurs.

Exemple 1:

entree: [1,2,4,3,-2,1,2]
sortie: -2

Exemple 2:

entree: [1]
sortie: 1

Signature:

def minimum(arr: List[int]): ->int

Constraints:

len(arr) > 1
