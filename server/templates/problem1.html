Factoriel

Description

Ecrire une function qui retourne Le factoriel d'un entier naturel n.

Exemple 1

Entree: 3
Sortie: 6
Expliquation: 3! = 1x2x3 = 6

Exemple 2

Entree: 5
Sortie: 120
Expliquation: 5! = 1x2x3x4x5 = 120

Type de la function

def factorial(n: int)--> int
Retourne un entier.
n: entier

Contraintes

n>= 1
